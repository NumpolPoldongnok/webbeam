🔥 เชื่อมอัตโนมัติ ไม่ต้องแลกรหัส · หลุดแล้วต่อกลับเอง (รัน broker บนเครื่องหลักเครื่องเดียว)

Broker IP

เครื่องหลัก: เปิด Terminal ไปที่โฟลเดอร์ WebBeam รัน node broker.mjs → มันจะบอก IP · ใส่ Broker IP = localhost แล้วกด "เริ่มอัตโนมัติ" · อีกเครื่อง: ใส่ IP ที่ broker บอก แล้วกด "เริ่มอัตโนมัติ" — เชื่อมเอง

✋ เชื่อมแบบ manual — เลือกบทบาท แล้วแลกรหัส/สแกน QR (ไม่ต้องมี broker)

1. สร้างรหัส → ส่งให้ B

2. วางรหัสตอบกลับจาก B → เชื่อมต่อ

1. วางรหัสจาก A → สร้างรหัสตอบกลับ

2. ส่งรหัสตอบกลับนี้ให้ A

IP เครื่องนี้ (LAN) ดูได้จาก System Settings → Wi-Fi → Details · ใส่ทั้งสองเครื่อง (IP ของใครของมัน) แล้วสร้างรหัสใหม่
